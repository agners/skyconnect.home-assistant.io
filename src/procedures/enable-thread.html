This guide assumes you have the following setup:

  • Connected SkyConnect.
  • Home Assistant OS installation.
  • Multiprotocol support disabled: Refer to the Disable multiprotocol support procedure.
  • This guide installs a firmware which supports only Thread! If you use your SkyConnect to control Zigbee devices, you will no longer be able to control them after installing the Thread firmware. Use a second SkyConnect or the Multiprotocol support (experimental).
  • Home Assistant Supervised: The Thread border router has some specific requirements on IPv6 configuration (e.g. IPv6 routing and prefix delegation support). This should be mostly met on a regular Debian installation, but this is not well tested by the developers. If you experience connectivity issue, try using a Home Assistant OS installation.
---

This guide outlines the step to install the OpenThread RCP (Radio Co-Processor) firmware for Thread only support. Following this guide will turn our Home Assistant installation into a a Thread border router which can be used to connect Matter over Thread devices to your network.
